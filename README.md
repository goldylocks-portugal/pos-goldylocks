## Overview

Goldylocks POS is a modern, cross-platform point-of-sale frontend that connects to the **Goldylocks ERP** ([www.goldylocks.pt](https://www.goldylocks.pt)), a Portuguese tax-compliant ERP platform. It runs on desktop (Windows, macOS, Linux), mobile (Android, iOS), and web.

---

## Features

- **Cross-platform** — works on desktop, mobile, and web
- **Receipt printing** — thermal printer support via Bluetooth and network
- **Barcode scanning** — fast item lookup with fuzzy search fallback
- **Offline-ready** — local database with automatic synchronization
- **ERP integration** — real-time sync with Goldylocks ERP ([www.goldylocks.pt](https://www.goldylocks.pt))
- **Tax compliance** — Portuguese tax regulations (e-invoicing, SAF-T)
- **Multi-language** — built-in support for multiple languages
- **Table management** — organize orders by table or takeaway
- **Responsive UI** — touch-friendly interface with dark theme
- **Bluetooth connectivity** — printer and peripheral support

## Quick Start

### Web Version

Open [pos.goldylocks.pt](https://pos.goldylocks.pt) in any modern browser.

### Desktop Version

1. Download the installer for your platform from the [releases page](https://github.com/goldylocks-portugal/pos-goldylocks/releases/latest).
2. Run the installer and follow the setup wizard.
3. Launch the application.

---

## Requirements

- **Desktop:** Windows 7+, macOS 10.13+, or Linux (x64)
- **Mobile:** Android 7+ or iOS 13+
- **Web:** Modern browser (Chrome, Firefox, Safari, Edge)
- **Bluetooth:** Required for peripheral connectivity features
